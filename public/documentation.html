Vid-Flix API

What It Does

This API interacts with a database that stores information on a large list of movies.

Routing

URL Endpoints Params Method Req Data Res Data
/movies Request, Response GET A large list of all movies to query A large list of all movies to query
/movies/:title Request, Response GET Fetches data of all of the movies within the data base Returns the movie list fetched and filters out the movie matching the title given at the end of the URL Endpoint
/movies/:title/genre Request, Response GET Fetches all the movies from the dtabase Filters through the list of movies fethed and returns the genre of the movie searched with a matching title
/movies/:title/directors Request, Response GET Fetches all the movies from the dtabase Filters through the list of movies fethed and returns the directors of the movie searched with a matching title
/register Request, Response POST Fetches user information If information value matches exsiting inormation the function will direct a new password, else, it will create a new user account on the website
/:user/settings Request, Response GET Fetches User account Provides access to user account personalization settings
/:user/favorites/add/:title Request, Response PUT Fetches a list of users favorite movies Adds a selected movie to the list of favorites by the movie title
/:user/favorites/delete/:title Request, Response DELETE Fetches a list of users favorite movies Deletes selected movie by title from users favorites list
/:user/settings/delete Request, Response DELETE Fetches all user account information Deletes all user account data
