Vid-Flix API

What It Does

This API interacts with a database that stores information on a large list of movies.

Routing

Description URL Endpoints Method Params Exmp Req Exmp Res
Returns an array of movies with title genre and much more information about them /movies GET Object /movies [ { title: 'Moana, genre: 'Adventure' }, { title: 'Superman', genre: 'Action' } ]
Returns a movie object which key 'title' value matches the req paramiters. /movies/:title GET Object, String /movies/Superman { title: 'Superman', genre: 'Action' }
Returns a string with a value from the genre key in the movie object which is returned by title /movies/:title/genre GET Object, String /movies/Superman/gernre 'Action'
Returns an Array with a value/'s from the directors key in the movie object which is returned by title /movies/:title/directors GET Object, Array /movies/Superman/directors ['Richard Donner']
Serves an html page with a form to register as a new user, or sign in if credentials exsist. /register POST Object /register { userName: '', pass: '' }
This endpoint will serve an html page to the user that shows a list of settings that can be customized /:user/settings GET Object, Boolean /JohnDoe/settings { email: false, pushNotifcation: true, showInfo: false }
This endpoint updates an array that contains all of the users favorite movies /:user/favorites/add/:title PUT Array, Object /JohnDoe/favorites/add/Superman [ { title: 'Superman', genre: 'Action' } ]
This endpoint deletes a favorite movie previously added to the Array of objects of favorite movies that matches the title given in the URL endpoint. /:user/favorites/delete/:title DELETE Object, String JohnDoe/favorites/delete/Superman 'Superman was deleted', { title: 'Superman', genre: 'Action' }
This endpoint deletes the users account completely. /:user/settings/delete DELETE HTML home page, string /JohnDoe/settings/delete 'Your account was succesfully deleted.'
